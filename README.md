UK Housing Market Analysis: Price Dynamics and COVID Shock
(SQL + Tableau)

Project Overview
This project analyzes UK housing transaction data to explore price patterns, property types, resale behavior, and housing market resilience during the COVID-19 shock of 2020. The analysis was conducted using SQL for data cleaning and transformation and Tableau for visualization.

Tools

SQL (data cleaning, CTEs, window functions)
Tableau (data visualization)
GitHub (project documentation)
Key Analytical Questions

How do housing prices differ across property types?
Do freehold properties show a price premium over leasehold?
How often are properties resold quickly?
How much do prices change between first and last sales?
How did housing prices behave during the 2020 COVID shock?
