UK Housing Market Analysis: Price Dynamics and COVID Shock
(SQL + Tableau)
Project Overview

This project analyzes UK housing transaction data to explore price patterns, property types, resale behavior, and housing market resilience during the COVID-19 shock in 2020.

The analysis combines SQL for data cleaning and transformation with Tableau for visualization, allowing deeper insight into how housing prices evolved across different property types and ownership structures.

The goal of the project is to demonstrate practical data analytics skills, including data cleaning, exploratory analysis, and visual storytelling.

Tools

SQL – data cleaning, CTEs, and window functions

Tableau – interactive data visualization and dashboards

GitHub – project documentation and version control

Key Analytical Questions

This analysis focuses on several questions about the UK housing market:

How do housing prices differ across property types (detached, semi-detached, flats, terraced)?

Do freehold properties show a price premium over leasehold properties?

How often are properties resold within short time periods?

How much do prices change between the first and last recorded sales?

How did housing prices behave during the 2020 COVID-19 shock?

SQL Analysis

The SQL analysis includes several data preparation and analytical steps:

Data cleaning and duplicate transaction removal

Date standardization for transaction timelines

Use of CTEs (Common Table Expressions) to structure queries

Window functions (LAG, LEAD) to track sequential property sales

Identification of property resale patterns

Calculation of price changes between sales

Example SQL concepts used in the project:

ROW_NUMBER() for deduplication

LAG() and LEAD() for sale comparisons

CTEs for modular query design
